## 🎓 Le Projet

Ce quizz est un projet que j'ai réalisé pour EVA Bordeaux Lac, dans un but de promotion du mode de jeu Color Chaos, c'est un quizz qui, grâce à des questions réponses, vous permet de savoir quel type de lapin vous êtes.

### 🛠️ Fonctionnalités

- **🚄 Résultat instantanné** : Le résultat de vôtre quizz est instantanné
- **❓ Questions** : Un total de sept questions
- **💡 Types de lapin** : Trois types : Chaos, Drama et Stratège
